#MySQL总结

1. 数据库中的数据类型

在数据库中，我们可以把数据分为：

  • 数值型
  • 字符型
  • 日期型
  • 布尔值

1.1 数值型

  • 整型：
    • tinyint 最小的整数，表示范围：-128~127，所占字节数：1 （是布尔值的底层构建）
    • smallint 小整数，表示范围：-32768~32767，所占字节数：2
    • mediumint 中整数，表示范围：-8388608~8388607，所占字节数：3
    • int 大整数，表示范围：-2147483648~2147483647，所占字节数：4
    • bigint 极大整数，表示范围：-9223372036854775808~9223372036854775807，所占字节数：8
  • 浮点数：
    • float 单精度浮点数，精度：6~7位，所占字节数：4
    • double 双精度浮点数，精度：15~16位，所占字节数：8
    • decimal(M,N) 定点数，精度：28位 ,M表示整数的位数，N表示小数的位数，所占字节数：无法确定(根据M和N来计算)

1.2 字符型：

  • 常用的字符串：
    • char(M) 定长字符，M表示字符的个数，最大长度：255，所占字节数：(也需要根据当前数据库的字符集去计算)
    • varchar(M) 变长字符(如果字符没到达M长度，便会以空格补全)，M表示字符的个数，最大长度：65535，所占字节数：(也需要根据当前数据库的字符集去计算)

1.3 日期型：

  • 日期：
    • date 日期，格式：YYYY-MM-DD，所占字节数：3
  • 时间：
    • time 时间，格式：HH:MM:SS，所占字节数：3
  • 日期时间：
    • datetime 日期时间，格式：YYYY-MM-DD HH:MM:SS，所占字节数：8
      在平时使用时，关于日期有一些常用的函数：
  • now() 获取当前的日期时间
  • year() 获取当前的年份
  • month() 获取当前的月份
  • day() 获取当前的日期
  • hour() 获取当前的小时
  • minute() 获取当前的分钟

数据库的操作

1. 数据库的创建、修改、删除

  • 创建数据库：
    • create database 数据库名;
  • 修改数据库：
    • alter database 数据库名 character set 字符集;
  • 删除数据库：
    • drop database 数据库名;(在日常使用时绝对不要用)
  • 查看数据库：
    • show databases;

2. 表的创建、修改、删除

  • 创建表：
    • create table 表名(
      字段名 字段类型,
      字段名 字段类型,
      …
      );
  • 修改表：
    • alter table 表名 modify 字段名 字段类型;(修改字段的数据类型)
    • alter table 表名 change 字段名 字段名 字段类型;(修改字段)
    • alter table 表名 add 字段名 字段类型;(添加字段)
    • alter table 表名 drop 字段名;(删除某个字段)
  • 删除表：
    • drop table 表名;
  • 查看表：
    • show tables;
